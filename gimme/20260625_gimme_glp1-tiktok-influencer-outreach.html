This report extends the existing GLP TikTok affiliate source list with 18 lookalike creators for Gimme Care's compounded semaglutide/tirzepatide affiliate program. Research pulled public TikTok profile stats and top-video snapshots where accessible (2026-06-25).

Patterns from existing top affiliates

  • Engagement beats follower count — @kekes.plot (195K followers, 20.4M profile likes) and @nfield94 set the benchmark.
  • Journey + check-in format wins — weekly weigh-ins, shot-day updates, what-I-eat-on-GLP (Ana, Johanna, Lily).
  • Polished but authentic — clean camera/lighting with relatable mom/lifestyle framing outperforms low-quality video.
  • Inquisitive comments — ideal audiences ask dosage, food, and side-effect questions (not spam).
  • Nurse/medical credibility — Lily (@lilianimariee) and nurse educators convert trust efficiently.
